Newton's First Law Of Motion: Linear Ramp

rho = ramp; theta = 0;

We use 10 particles and render (rho, theta) as Polar coordinates. They collide together so you can only see 1 particle.

rho

Particle ID and Polar Coordinates

rho = ramp, theta = id * 36.

To split up the 10 particles, we can use the particle ID.

To Be, Or Not To Be, That Is Not A Question

rho = ramp | window(10), theta = id * 36.

Particles should be able to die and reborn. Every 10 ticks (1 tick = 0.1 second) a particle dies.

rho

Butter Smooth Motion

rho = ramp | easeout(10) | window(10), theta = id * 36.

Non-linear movement looks better.

easeout(10)

Remember that timeline must be defined from -infinity to +intinity? When x < 0, we define easeout(10) to be 0.

rho
